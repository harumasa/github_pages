How to install Arch Linux

August 29, 2020 (Lastmod: Jul 24 2022)

Introduction

  • Model: DELL Inspiron 14 (5485)
  • OS: Arch Linux

Installation guide

  • Installation guide

1. Pre-installation

1.1 Acquire an installation image

  • Arch Linux Downloads

1.2. Verify signature

gpg --keyserver-options auto-key-retrieve --verify archlinux-version-x86_64.iso.sig

1.3. Prepare an installation medium

dd bs=4M if=path/to/archlinux-version-x86_64.iso of=/dev/sdx conv=fsync oflag=direct status=progress

1.4. Boot the live environment

Set BIOS configuration: Press F2

  • Sleep Mode: Force S3
  • Fast Boot: Thorough (Disabled)
  • Secure Boot: Disabled

Boot from the USB flash drive: Press F12

1.5. Set the console keyboard layout

loadkeys jp106

1.6. Verify the boot mode

ls /sys/firmware/efi/efivars

1.7. Connect to the Internet

iwctl

[iwd]# device list
[iwd]# station ${DEVICENAME} scan
[iwd]# station ${DEVICENAME} get-networks
[iwd]# station ${DEVICENAME} connect SSID
[iwd]# exit

ping archlinux.org

1.8. Update the system clock

timedatectl set-ntp true
timedatectl status

1.9. Partition the disks

Identify devices

fdisk -l

Example layout (UEFI with GPT)

Mount point: Partition: Partition type: Size:
/mnt/boot /dev/nvme0n1p1 EFI system partition 512 MB
/mnt /dev/nvme0n1p2 Linux filesystem Remainder of the device

Create partitions

cgdisk /dev/nvme0n1

1. 512MB EFI partition # Hex code ef00
2. 100% size partiton # Hex code 8300

Encrypting devices

cryptsetup luksFormat /dev/nvme0n1p2
cryptsetup open --type luks /dev/nvme0n1p2 cryptroot

Create encrypted partitions

pvcreate /dev/mapper/cryptroot
vgcreate vg0 /dev/mapper/cryptroot
lvcreate -l +100%FREE vg0 -n root

1.10. Format the partitions

mkfs.fat -F 32 /dev/nvme0n1p1
mkfs.ext4 /dev/mapper/vg0-root

1.11. Mount the file systems

mount /dev/mapper/vg0-root /mnt
mkdir /mnt/boot
mount /dev/nvme0n1p1 /mnt/boot
lsblk /dev/nvme0n1

2. Installation

2.1. Select the mirrors

nano /etc/pacman.d/mirrorlist
  • Pacman Mirrorlist Generator

2.2. Install essential packages

pacstrap /mnt base base-devel linux linux-firmware nano

3. Configure the system

3.1. Fstab

genfstab -U /mnt >> /mnt/etc/fstab

3.2. Chroot

arch-chroot /mnt

3.3. Time zone

ln -sf /usr/share/zoneinfo/Asia/Tokyo /etc/localtime
hwclock --systohc --utc

3.4 Localization

Uncomment locales

nano /etc/locale.gen

Uncomment line en_US.UTF-8 UTF-8
Uncomment line ja_JP.UTF-8 UTF-8

Generate locales

locale-gen

Set the language and keyboard layout

echo LANG=en_US.UTF-8 > /etc/locale.conf
echo KEYMAP=jp106 > /etc/vconsole.conf

3.5. Network configuration

Set the hostname

echo ${MYHOSTNAME} > /etc/hostname

Add matching entries

nano /etc/hosts

127.0.0.1 localhost
::1       localhost
127.0.1.1 ${MYHOSTNAME}

Setup NetworkManager

pacman -S networkmanager dhcpcd
systemctl enable NetworkManager

3.6. Initramfs

Modify mkinitcpio.conf

nano /etc/mkinitcpio.conf

Add 'ext4' to MODULES
Add 'encrypt' and 'lvm2' to HOOKS before 'filesystems'

Recreate the initramfs image

pacman -S lvm2
mkinitcpio -p

3.7. Root password

passwd

3.8. Boot loader

Install systemd-boot

bootctl install

Setup systemd-boot

nano /boot/loader/loader.conf

default arch.conf
timeout 4
console-mode max
editor  no

Adding loaders

nano /boot/loader/entries/arch.conf

title   Arch Linux
linux   /vmlinuz-linux
initrd  /amd-ucode.img
initrd  /initramfs-linux.img

Setup Microcode

pacman -S amd-ucode

4. Reboot

exit
umount -R /mnt
reboot

5. Post-installation

Login with root account

Setup NetworkManager

nmtui

General recommendations

1. System administration

1.1. Users and groups

Login with root account

useradd -m -g users -G wheel ${USERNAME}
passwd ${USERNAME}

To allow members of group wheel sudo access

EDITOR=nano visudo

Uncommment line %wheel ALL=(ALL) ALL
Uncommment line %wheel ALL=(ALL) NOPASSWD: ALL

Return to regular user

exit

2. Package management

2.1. pacman

Update all packages

sudo pacman -Syu

2.5. Arch User Repository

Setup paru (AUR Helper)

sudo pacman -S --needed base-devel
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si

Utilizing multiple cores on compression

sudo nano /etc/makepkg.conf

Change line COMPRESSXZ=(xz -c -z --threads=0 -)

Update all packages

paru
